# practice-project

An open source clone of the popular Stack game for iOS and Android

## Installation

```bash
$ npm install
$ npm install -g typescript
$ npm install -g gulp
$ npm start
```

## Production Build
```bash
$ npm run build
```

## Producton Runtime
```bash
$ node app.js -p
```
